Help

This is an example of ADF application creating processes and tasks for Alfresco Process Services.

The application assumes you have an Alfresco Process Services up an running.

Choose first if you want to create a new process or a new task.

If you decide to create a process, select an App first and then click on the CREATE PROCESS button. Once filled the form, press the START button. To check the new process, access to the Alfresco-Activiti User Interface and you will see the new process.

If you decide to create a task, press the CREATE TASK button. Once filled the form, press the START button. To check the new task, access to the Alfresco-Activiti User Interface and you will see the new task.
